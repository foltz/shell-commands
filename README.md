# Shall Commands
Dotfiles for Simplified CLI

## Installation

- checkout this repo into your ~/Dev/ folder
- add these lines to your `.bash_profile` or `.zprofile` file:

```
source ~/Dev/shell-commands/git
source ~/Dev/shell-commands/other
```
